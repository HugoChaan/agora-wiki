# Agora Intake

Run this when the user request does not clearly identify a single Agora product or when the request spans multiple products.

## Classify into
- RTC
- RTM
- Conversational AI
- Cloud Recording
- Server / token generation
- Server Gateway
- multi-product integration

## Then output
- selected products
- target platforms
- auth implications
- live-doc requirements
- next product files to load
