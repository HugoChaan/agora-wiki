# Agora Intake

Use this skill when the request does not clearly identify a single Agora product or when it spans multiple products.

- RTM
- Conversational AI

- Server / token generation
- Server Gateway
- multi-product integration

## Then output

- target platforms
- auth implications
- live-doc requirements
- next Agora sub-skills to load
